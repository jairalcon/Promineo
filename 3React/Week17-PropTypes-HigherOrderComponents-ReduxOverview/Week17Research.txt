What is TypeScript and why do some companies use it?
TypeScript has been built on JavaScript to add better tools to assist programmers with TypeScript's type system. TypeScript can pinpoint values that might have typos or incorrect variables usage, thus minimizing the occurrences of bugs.
For example, in JavaScript, a variable is assigned a variable type such as string, number. TypeScript will use the variable value as it's associated type.

When should you use Redux?
We should use Redux when there are large amounts fo app state that are needed many places within the app, the state is updated regularly, when the app becomes more large-scale with many developers working on app, or when you need to see how the state is updated throughout using the app.
